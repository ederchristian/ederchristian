## Getting Started

### Installation

```sh
git clone https://github.com/ederchristian/ederchristian.git && cd ederchristian && npm install
```

or

```sh
git clone https://github.com/ederchristian/ederchristian.git && cd ederchristian && yarn
```

### Basic Usage

Running it:

```sh
npm run dev
```

or

```sh
yarn dev
```

In your browser, go to:

```sh
http://localhost:4321/
```
